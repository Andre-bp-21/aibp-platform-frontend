# AIBP Platform - Frontend

Frontend profesional de la Plataforma Adaptativa de Inteligencia Empresarial.

## 🚀 Características

- Dashboard interactivo
- Autenticación de usuarios
- Análisis empresarial con IA
- Simulación de escenarios
- Subida de archivos Excel/CSV
- Gráficos en tiempo real

## 🛠️ Tecnologías

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Recharts
- Zustand

## 📋 Requisitos

- Node.js 18+
- npm o yarn

## 🏃 Inicio Rápido

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Compilar para producción
npm run build

# Iniciar en producción
npm start
